Atlas Agent

Commands
- atlas : Atlas Agent가 지원하는 명령어를 설명합니다.
- help : Atlas Agent가 지원하는 명령어를 설명합니다.
- versions: Atlas Agent가 추천하는 버전을 출력합니다.
- docs: Atlas Agent가 불러올 수 있는 문서 목록을 출력합니다.
- doc: Atlas Agent가 문서를 불러옵니다.
	- query: 해당하는 키워드가 포함된 문서를 불러옵니다.
	- mode(preview / full): 해당하는 문서를 조회하는 방식을 선택합니다.
		- preview: 문서 일부를 출력합니다.
		- full: 문서 전체를 출력합니다.
- search: Atlas Agent가 키워드가 포함된 문서를 탐색합니다.
	- query: 키워드를 입력합니다.
- nest-cli: Nest CLI 명령어를 출력합니다.
